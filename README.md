# Generative-AI-Chatbot-using-Google-Gemini
Developed an AI-powered chatbot using the Google Gemini API. Built an interactive web interface with Streamlit. Enabled users to ask natural language questions and receive AI-generated responses in real time. Integrated the Gemini Large Language Model (LLM) for conversational AI.

---

## 📌 Project Overview

This project demonstrates how to integrate the Google Gemini API into a Streamlit application to create an interactive AI chatbot. It provides a clean user interface where users can enter prompts and instantly receive AI-generated responses.

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/Generative-AI-Chatbot-using-Google-Gemini.git
```

### 2. Navigate to the project folder

```bash
cd Generative-AI-Chatbot-using-Google-Gemini
```

### 3. Install the required libraries

```bash
pip install -r requirements.txt
```

### 4. Configure your Gemini API Key

Do **not** hardcode your API key in the source code.

Set it as an environment variable:

```python
import os

os.environ["GEMINI_API_KEY"] = "YOUR_API_KEY"
```

### 5. Run the application

```bash
streamlit run app.py
```

## 💻 How It Works

1. User enters a question.
2. Streamlit sends the prompt to the Google Gemini model.
3. Gemini processes the prompt.
4. The AI-generated response is displayed instantly on the webpage.
